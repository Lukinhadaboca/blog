Você sabe o que é e como funciona a Deep Web?

E aí pessoal, tudo bem? Espero que estejam prontos para tratarmos sobre o assunto do site misterioso DEEP WEB. Nesse site achamos muitas coisas ruins, muitas coisas boas e perturbadoras. Esse site e encontrado com facilidade no Google, Bing e Yahoo. Não existe uma forma única de acessar a internet profunda. Isso porque a deep web é composta por várias redes separada, que não conversam entre si. A rede mais conhecida é o TOR, software de anonimato de código aberto, que frequentemente aparece na mídia. Entre as páginas conhecidas está o SILK ROAD, um site de venda de drogas derrubado pela FBI, mas que rapidamente ressurgiu em outros endereços

Vamos ensinar como entrar na deep web, mas deixamos alerta que pode ser perigoso:

Passo 1: Baixe o TOR

O primeiro passo é baixar o Tor, que possui versões para Windows, Mac e Linux. A maneira mais prática de começar a acessar os sites da deep web é baixar o Tor Browser, que traz um pacote completo com software que se conecta à rede Tor e um navegador pré-configurado para acessar a rede. Você pode baixar o Tor browser na página oficial do Tor Project (torproject.org).

Passo 2: Execute o instalador

Na sequência, basta executar o instalador e seguir as instruções na tela. É provável que o assistente pergunte se você possui uma internet "livre de obstáculos"; a não ser que você esteja em uma rede censurada ou filtrada, basta clicar no botão "Connect" e o navegador será aberto imediatamente. Você está pronto para acessar a deep web

Passo 3: Acesse o diretório de links

Para acessar os sites escondidos na deep web, é preciso entender que não é possivel encontrá-los em um buscador convencional, como google. Por isso, o ponto de partida mais comum e procurar um diretorio de sites. No Tor, os links são formados por caracteres aleatórios seguidos da terminação .onion. O diretório mais famoso do Tor é a HIDDEN WIKI. Pelo Tor Browser, abra o endereço.p

Finalidade

A finalidade da Deep Web é sempre um assunto polêmico, vou focar nas coisas boas, ademais, as ruins nem podemos considerá-las como sendo “finalidades”.

A primeira coisa boa da Deep Web é certamente a possibilidade de se compartilhar e hospedar arquivos sigilosos que normalmente jamais poderiam estar disponíveis na surface.

A segunda coisa boa é a liberdade de expressão. Em países onde a liberdade do povo é violada o governo começa todo o controle do que pode ou não ser visto pela internet. Isso é feito para que o mundo não saiba o seu modus operand.

O anonimato que a Deep Web oferece faz com que as pessoas dentro desses tipos de países possam relatar o que acontece lá e até mesmo pedir por socorro. Veja que as informações que vimos acima não são totalmente exatas, afinal, estamos falando de um assunto que muda constantemente.

Para que você tenha uma ideia, em 2015 o Google relatou que seu buscador contava com mais de 14,5 bilhões de paginas indexadas. Hoje esse número deve ter subido muito mais, imagine: se na surface temos tantas páginas, estima-se que na Deep Web existam trilhões. É certo que isso é apenas uma especulação, afinal, como na Deep Web não temos páginas indexadas fica muito difícil saber seus números exatos.

Como a Deep Web é dividida?

É comum os sites falarem que a divisão da Deep Web se dá em camadas. Isso é um erro grave. Ela é dividida em redes, sendo a Onion e a I2P, que são as mais conhecidas.

Não se têm camadas de profundidade com graus de dificuldade para acesso, são redes com praticamente o mesmo grau de dificuldade.

Posso usar o meu navegador de internet para acessar a Deep Web?

Não, de verdade! A Deep Web não pode ser acessada com os navegadores que estamos acostumados a usar. Para acessá-la temos softwares específicos e um dos mais utilizados é o Tor.
